# Otimizador-de-Carteira-de-Investimentos
Sistema em C/C++ que otimiza e rebalanceia carteiras de investimento. Utiliza Árvores N-árias para estruturar categorias e ativos, calculando percentuais recursivamente. Detecta desbalanceamento e sugere compras/vendas automáticas para manter os perfis (Conservador, Moderado, Arrojado) conforme o alvo definido.
